Stock Data Server

Stock Data Server is a full-stack web application that allows users to retrieve and visualize historical stock data by entering any valid ticker symbol. The app connects to the Yahoo Finance API to fetch the past 30 days of closing prices and corresponding dates, displaying the data in a clean, readable format. Built with a Node.js/Express backend to handle API requests and a React frontend for user interaction, the project demonstrates proficiency in third-party API integration, asynchronous data fetching, and full-stack application architecture.
